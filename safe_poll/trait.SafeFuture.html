An equivalent to Future which returns SafePoll instead of Poll.

Returning SafePoll enforces an extra constraint where a previously created WakeupRegisteredToken must be used. This extra constraint helps to verify that the SafeFuture was properly registered for a wakeup in order to be polled again later. It's possible to fulfill that constraint either by having an internal type return SafePoll::Pending with a token that can be reused or by manually creating the token inside an unsafe block.
